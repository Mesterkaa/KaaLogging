How to use

First you need to login via Google in the top right corner
Go to your profile page a check your KaaToken, which is used to identify you with your logs.

Get Logs

To get all your logs send a GET request to this URL or go to the GetLogs section in the top menu

http://Mesterkaa.hopto.org:1337/GetLogs?KaaToken=YOUR_KAATOKEN

Per standard all your logs will be send back to you. If you want to filter the logs, use the Category and Title paramenters

KaaToken: YOUR_KAATOKEN : Required!
Category: CATEGORY_NAME : Optional
Title: TITLE_NAME : Optional

Post Logs

To create a log send a POST request to this URL or go to the PostLogs section in the top menu

http://Mesterkaa.hopto.org:1337/PostLogs

To specifiy Category, Title and Content use the following paramenters

KaaToken: YOUR_KAATOKEN : Required!
Category: CATEGORY_NAME : Optional
Title: TITLE_NAME : Optional
Content: CONTENT : Optional
